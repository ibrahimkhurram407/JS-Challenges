Grid

Grid is a layout system that allows you to create complex layouts with rows and columns. Here are some of the most common style attributes:

  • display: set to grid to enable grid layout.
  • grid-template-columns: specifies the width of each column.
  • grid-gap: specifies the space between grid items.
  • justify-items: specifies how grid items are aligned horizontally.
  • align-items: specifies how grid items are aligned vertically.

Flexbox

Flexbox is a layout system that allows you to easily align and distribute content within a container. Here are some of the most common style attributes:

  • display: set to flex to enable flexbox layout.
  • justify-content: specifies how content is aligned horizontally within the container.
  • align-items: specifies how content is aligned vertically within the container.
